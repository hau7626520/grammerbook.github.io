# 英文文法武功祕笈

歡迎來到「英文文法武功祕笈」！這是一個旨在幫助您系統學習和掌握英文文法的互動式學習資源。我們將複雜的文法知識拆解成易於理解的章節，並輔以視覺化圖解與互動測驗，助您打下扎實的英文基礎。

## 專案特色

*   **系統化學習路徑**：內容依章節精心編排，從句子的基礎到名詞的深入探討，循序漸進。
*   **視覺化教學卡片**：透過圖解卡片，將抽象的文法概念轉化為直觀的圖像，加深理解。
*   **互動式武功測驗**：每節課程後皆附有測驗題，即時檢驗學習成效，鞏固所學。
*   **清晰易用的導覽**：左側導覽列讓您輕鬆穿梭於各章節之間，隨時回顧或預習。
*   **輕量化設計**：作為純靜態網頁，無需安裝任何伺服器，直接在瀏覽器中即可開啟學習。

## 如何使用

1.  **開啟檔案**：在您的網頁瀏覽器中打開 `1-1-1.html` (或其他任何 `.html` 檔案)。
2.  **開始學習**：您會看到左側的導覽列，列出了所有可用的章節和單元。點擊任何連結即可開始您的文法學習之旅。
3.  **進行測驗**：在每個單元下方，您可以找到「武功測驗」區塊，點擊選項作答並查看詳解。

## 專案結構 (部分)

*   `*.html`：各個文法單元的獨立頁面。
*   `navigation.js`：負責生成和管理左側導覽列的JavaScript檔案。
*   `under_construction.html`：尚未完成的頁面會連結到此，提示該內容正在建置中。
*   `README.md`：您正在閱讀的專案說明文件。
